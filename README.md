Меня зовут Лаура Арыстанагали, и я хочу рассказать о том, как я создала свой проект — «Louro CRM».В процессе разработки моим главным помощником стал искусственный интеллект Gemini. Он помог мне с технической базой, но чтобы превратить программный код в реальный инструмент для бизнеса, я внесла много авторских изменений.
Прежде всего, я переработала главную страницу: добавила удобное меню, чтобы навигация по разделам была мгновенной и понятной. Также я адаптировала систему под конкретные рабочие задачи — теперь при создании заказа можно вводить полное ФИО и ИИН клиента. Это позволяет вести базу данных профессионально и точно. Кроме того, я реализовала функцию поиска по ИИН, благодаря чему найти любую информацию о клиенте или заказе теперь можно в пару кликов.
«Louro CRM» для меня — это проект, в котором современные технологии ИИ соединились с моими идеями для решения реальных задач предпринимателей.
